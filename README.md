# Dashboard de Análise de Futebol — Power BI

Dashboard interativo de análise de desempenho de jogadores de futebol,
desenvolvido como projeto prático de Business Intelligence.

## Sobre o projeto

Análise dos dados de 20 jogadores de 5 times do Campeonato Brasileiro,
com foco em métricas de desempenho ofensivo por jogador e por time.

## Visuais criados

- Cartões de KPI: Total de Gols, Total de Assistências e Média de Gols
- Gráfico de barras: Gols por Time
- Tabela detalhada: desempenho individual de cada jogador
- Segmentação de dados: filtro interativo por time

## Métricas DAX criadas

- Total de Gols
- Total de Assistências
- Média de Gols
- Participações em Gols (gols + assistências)

## Ferramentas utilizadas

- Power BI Desktop
- Power Query (limpeza e transformação de dados)
- DAX (criação de medidas)
- Excel (fonte de dados)
